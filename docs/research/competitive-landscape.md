# Pilot — Competitive Landscape

## Direct Competitors

### Anthropic Computer Use
- **What:** API feature that lets Claude control a computer via screenshots + actions
- **Target:** Developers building automation tools
- **Pricing:** API usage costs ($3-15/MTok)
- **Limitation:** Developer tool, not a consumer product. Requires API key, coding knowledge.
- **Our advantage:** We're the consumer product this should have been.

### OpenAI Operator
- **What:** Cloud-based agent that browses the web on your behalf
- **Target:** ChatGPT Pro subscribers
- **Pricing:** Included in ChatGPT Pro ($200/mo)
- **Limitation:** Cloud-only, can't access local apps, limited to web browser, expensive tier.
- **Our advantage:** Local, works with any app (not just browser), uses user's existing AI.

### Adept AI (ACT-1)
- **What:** AI agent that operates software through natural language
- **Target:** Enterprise
- **Pricing:** Not publicly available (enterprise sales)
- **Status:** Raised $415M, reportedly acquired/acqui-hired by Amazon
- **Limitation:** Vaporware for consumers. Enterprise-only.
- **Our advantage:** Actually shipping to consumers.

### Rewind.ai / Limitless
- **What:** Records everything on screen, makes it searchable
- **Target:** Knowledge workers
- **Pricing:** $20/mo
- **Limitation:** Only recalls, doesn't ACT. Can search "what was that website?" but can't go back and fill in the form.
- **Our advantage:** We act, they watch.

### Apple Intelligence
- **What:** Built-in AI features in macOS/iOS
- **Target:** All Apple users
- **Pricing:** Free (built into OS)
- **Limitation:** Very conservative capabilities. Writing tools, image generation, basic Siri improvements. Won't operate apps on your behalf.
- **Our advantage:** Actually autonomous. Does things, not just suggests.

### Rabbit R1
- **What:** Dedicated hardware device with "Large Action Model"
- **Target:** Consumers
- **Pricing:** $199 device
- **Status:** Largely failed — reviews terrible, returned to focusing on software
- **Limitation:** Separate device nobody wants to carry. Limited app support.
- **Our advantage:** Software on the device you already have.

## Adjacent / Partial Competitors

### Zapier / Make / n8n
- Workflow automation via APIs
- Requires technical setup, API connections
- Can't handle anything without an API
- **Our advantage:** Works with anything visual, no setup

### Keyboard Maestro / Alfred / Raycast
- Mac automation via hotkeys and scripts
- Powerful but requires manual programming
- **Our advantage:** Natural language, AI-driven, learns automatically

### UI.Vision / Selenium
- Browser automation via recording
- Brittle, breaks when UI changes
- **Our advantage:** Vision-based, adapts to changes

## Market Positioning

Pilot sits in a unique space: **consumer-grade AI automation that requires zero technical knowledge and zero cloud infrastructure.**

No one else offers:
- Local-only operation
- BYOB (Bring Your Own Brain) model
- Zero API keys
- Works with any desktop app, not just browsers
- Privacy by architecture (not by policy)

## Market Size

- 100M+ Mac users worldwide
- 10M+ paying ChatGPT subscribers
- 5M+ paying Claude subscribers
- TAM for "AI assistant that actually does stuff": ~$10B by 2028
- Realistic year 1 target: 10,000 paying users × $100/yr = $1M ARR
